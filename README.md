1. Clone repository

3. Add Environment Variables

```shell
REACT_APP_GOOGLE_MAPS_API_KEY=<YOUR GOOGLE MAPS API KEY>

```

4. Run Application

```shell
$ npm start
```

or

```shell
$ yarn start
```

4. Deployment

```shell
$ npm run build
```

or

```shell
$ yarn build
```
